## üåü Key Features

* **Orbital Mechanics**: Dynamic planet rotation with orbital decay and moon spawning systems.
* **Combat System**: Physics-based "Spike & Chain" mechanics for grappling and attacking.
* **Multiplayer**: Fully implemented split-screen multiplayer support with lobby and join mechanics.
* **Visuals**: Custom sprite work, normal mapping, and dynamic lighting effects.
* **UI/UX**: Comprehensive HUD, minimap, score systems, and polished menus.

## üöÄ Getting Started

### Prerequisites

* **Unity Hub**
* **Unity Editor** (Check `GamesFleadhProject/ProjectSettings/ProjectVersion.txt` for the exact version)

### Installation

1.  Clone the repository:
    ```bash
    git clone [https://github.com/SamsonA00321296/Games-fleadh.git](https://github.com/SamsonA00321296/Games-fleadh.git)
    ```
2.  Open **Unity Hub**.
3.  Click **Add** and select the `GamesFleadhProject` folder (Note: The project root is inside this subfolder).
4.  Launch the project.
5.  Open the **Title Scene** to start the game flow.
